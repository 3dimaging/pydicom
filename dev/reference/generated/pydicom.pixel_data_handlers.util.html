pydicom.pixel_data_handlers.util.convert_color_space(arr, current, desired)[source]¶

Convert the image(s) in arr from one color space to another.

Parameters

  • arr (numpy.ndarray) – The image(s) as a numpy.ndarray with shape (frames, rows, columns, planes) or (rows, columns, planes).

  • current (str) – The current color space, should be a valid value for (0028,0004) Photometric Interpretation. One of 'RGB', 'YBR_FULL'.

  • desired (str) – The desired color space, should be a valid value for (0028,0004) Photometric Interpretation. One of 'RGB', 'YBR_FULL'.

Returns

The image(s) converted to the desired color space.

Return type

numpy.ndarray
